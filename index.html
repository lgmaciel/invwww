A invenção da WEB

Tim Berners-Lee, cientista britânico, inventou a World Wide Web (WWW) em 1989, quando trabalhava no CERN (Conseil européen pour la recherche nucléaire).

Este é Tim Berners-Lee, criador da WWW.

A web livre

Em 30 de abril de 1993 o CERN publicou uma declaração disponibilizando livremente tecnologias básicas necessárias para construír a World Wide Web, permitindo que a web florescesse.

Captura da tela do browser original do NeXT em 1993 (Imagem: Berners-Lee/CERN)

As tecnologias disponibilizadas pelo CERN foram:

  • Um cliente (browser) básico - veja mais aqui
  • Um servidor web básico
  • biblioteca com código comum aos projetos www
